🎃 Cartwright Ranch Trick or Treat Map 🍬

Click on the map to mark if you're handing out candy!

🎃 = Giving out candy!

How to use:

  1. Click the "Mark My House" button to enable marking mode
  2. Click on your house location on the map grid
  3. Your house will be marked with a 🎃 icon
  4. Click on any existing marker to remove it
  5. Use zoom buttons to adjust the view

Note: Markers are now shared across all users in real-time!

⚠️ Configuration Required

Please add your Firebase and Google Maps API keys to app.js to enable shared markers and map view.
